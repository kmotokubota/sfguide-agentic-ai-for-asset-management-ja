## エグゼクティブリーダーシップ

### Executive Copilot - 包括的ビジネスパフォーマンスレビュー

#### ビジネスコンテキスト設定

**ペルソナ**: Sarah Chen、Simulated Asset Managementのアセットマネジメント責任者  
**ビジネス課題**: 経営委員会や取締役会向けに、会社全体のパフォーマンス、クライアントフロー、戦略的方向性の包括的な可視性が必要です。  
**価値提案**: 会社全体KPI、クライアント分析、戦略ドキュメントにまたがるシングルペインオブグラスビュー。

**エージェント**: `executive_copilot`

#### デモフロー

##### PDF付き完全な取締役会ブリーフィング

**ユーザー入力**: 
```
取締役会向けに、すべての戦略にわたる会社全体のAUMとパフォーマンス、集中懸念を含むクライアントフロー分析、コンテキスト付きの最高および最低パフォーマンス戦略、現在の戦略的方向性を説明する関連する投資哲学ポジショニングをカバーする完全なエグゼクティブブリーフィングを準備してください。取締役会配布用のプロフェッショナルなPDFを生成してください。
```

**使用ツール**:
- `executive_analyzer` (Cortex Analyst) - 会社全体KPI
- `quantitative_analyzer` (Cortex Analyst) - 戦略パフォーマンス
- `search_strategy_documents` (Cortex Search) - 戦略ドキュメント
- `pdf_generator` - 取締役会資料

**期待される応答**:
- 会社全体AUMサマリー
- 戦略別パフォーマンス
- クライアントフロー分析
- トップ/ボトム戦略とコンテキスト
- 戦略的方向性ハイライト
- 取締役会配布用PDF

### Executive Copilot - 戦略的競合分析＆M&Aシミュレーション

##### M&Aシミュレーション

**ユーザー入力**: 
```
これをモデル化しましょう。500億ドルAUMのビジネスを買収する高レベルのシミュレーションを実行してください。見つけた収益数値を使用し、コストシナジーに関する当社の標準的な前提を適用してください。初年度の1株当たり利益への予想される影響は何ですか？
```

**使用ツール**:
- `sec_financials_analyzer` (Cortex Analyst) - 競合財務データ
- `ma_simulation_tool` - M&A財務モデリング
- `search_press_releases` (Cortex Search) - 競合ニュース

**期待される応答**:
- ターゲット企業財務サマリー
- 収益/コストシナジー前提
- EPS影響シミュレーション
- 統合リスクと機会
- 推奨アクション

#### シナリオラップアップ

**ビジネスインパクトサマリー**:
- 経営委員会準備時間の大幅短縮
- SEC提出書類からのリアルタイム競合インテリジェンス
- 自動EPSアクリーションモデリング付きM&Aシミュレーション
- プロフェッショナルな取締役会資料の即座生成
