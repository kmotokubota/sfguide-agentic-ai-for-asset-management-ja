# SAMデモ - リスク＆コンプライアンスシナリオ

リスク＆コンプライアンスロール向けの完全なデモシナリオ。

---

## リスク＆コンプライアンス

### ESG Guardian - ESGリスク監視＆ポリシーコンプライアンス

#### ビジネスコンテキスト設定

**ビジネス課題**: ESGリスクの継続的監視とポリシーコンプライアンスの確保が必要です。  
**価値提案**: プロアクティブなコントロバーシースキャン、エンゲージメント履歴レビュー、ポリシーコンプライアンス評価を組み合わせた包括的ESGガバナンス。

**エージェント**: `esg_guardian`

#### デモフロー

**ユーザー入力**: 
```
当社のESGポートフォリオに影響を与える可能性のあるコントロバーシーをスキャンし、ポリシー閾値との比較、エンゲージメント履歴を含む完全なリスク評価を提供してください。
```

**期待される応答**:
- 検出されたコントロバーシー（深刻度別）
- ポートフォリオインパクト分析
- ポリシー閾値との比較
- エンゲージメント履歴サマリー
- 推奨アクション

---

### Compliance Advisor - マンデート監視＆違反検出

#### ビジネスコンテキスト設定

**ビジネス課題**: すべてのポートフォリオとマンデートにまたがる継続的なコンプライアンス監視が必要です。  
**価値提案**: ポリシードキュメント、是正計画、監査対応ドキュメントを含む自動違反検出。

**エージェント**: `compliance_advisor`

#### デモフロー

**ユーザー入力**: 
```
すべてのポートフォリオのマンデートコンプライアンスステータスをチェックし、違反、警告、是正が必要な項目をハイライトしてください。
```

**期待される応答**:
- ポートフォリオ別コンプライアンスステータス
- 違反と警告のサマリー
- ポリシー閾値との詳細比較
- 是正推奨とタイムライン

#### シナリオラップアップ

**ビジネスインパクトサマリー**:
- 複数NGOソースにまたがるプロアクティブESGリスク監視
- 包括的エンゲージメント追跡とコミットメント監視
- ポリシーベースのアクションガイダンス付き自動コンプライアンスチェック
- すべてのポートフォリオとマンデートにまたがるリアルタイム監視
- 市場インパクト考慮を含む洗練された是正計画

**技術的差別化要因**:
- マルチソースESGインテリジェンス統合
- 自動ポリシー閾値チェック
- 監査対応ドキュメント自動生成
