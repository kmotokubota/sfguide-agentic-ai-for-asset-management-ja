## セールス

### Sales Advisor - 四半期クライアントプレゼンテーション

#### ビジネスコンテキスト設定

**ペルソナ**: James、Simulated Asset Managementのセールスマネージャー  
**ビジネス課題**: クライアント向け四半期レポートの作成には、パフォーマンスデータ、テンプレート、哲学、規制開示の統合が必要です。  
**価値提案**: 複数ソースを統合したプロフェッショナルなクライアントレポート自動生成。

**エージェント**: `sales_advisor`

#### デモフロー

**ユーザー入力**: 
```
[ポートフォリオ名]の四半期クライアントレポートを作成してください。パフォーマンス、上位保有銘柄、ESGメトリクス、投資哲学、規制開示を含めてください。
```

**期待される応答**:
- 四半期パフォーマンスサマリー
- ベンチマーク比較
- 上位保有銘柄
- セクター配分
- ESGメトリクス
- 投資哲学ハイライト
- 規制免責事項

### Sales Advisor - 新規クライアントオンボーディングワークフロー

#### ビジネスコンテキスト設定

**ビジネス課題**: 新規クライアントのオンボーディングには、ウェルカム資料、主要連絡先、投資ポリシー、次のステップの準備が必要です。

#### デモフロー

**ユーザー入力**: 
```
新規クライアント[クライアント名]のオンボーディングパッケージを準備してください。ウェルカム資料、主要連絡先、投資ポリシーサマリー、初期ミーティングアジェンダを含めてください。
```

**期待される応答**:
- パーソナライズされたウェルカムレター
- 主要連絡先情報
- 投資ポリシーサマリー
- オンボーディングスケジュール
- 初期ミーティングアジェンダ
