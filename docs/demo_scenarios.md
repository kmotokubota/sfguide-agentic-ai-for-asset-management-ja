# SAMデモ - シナリオスクリプト

ロールとエージェント別に整理された完全なデモシナリオ。ステップバイステップの会話、期待される応答、データフローを含みます。

## ロール別デモシナリオ

このドキュメントはビジネスロール別に整理されており、各ロールには専用のシナリオドキュメントがあります：

### [ポートフォリオマネージャーシナリオ](demo_scenarios_portfolio_manager.md)
**エージェント**: Portfolio Copilot、Thematic Macro Advisor

**シナリオ**:
- **Portfolio Copilot - ポートフォリオインサイトとベンチマーキング**: 保有銘柄レビューから統合リスク管理を伴う実行可能な投資決定までの完全なポートフォリオ分析ワークフロー
- **Portfolio Copilot - リアルタイムイベントインパクトと二次リスク検証**: 直接エクスポージャーとマルチホップサプライチェーン依存関係分析を組み合わせたイベント駆動型リスク評価
- **Portfolio Copilot - AI支援マンデートコンプライアンスと証券代替**: 違反検出から代替分析、委員会ドキュメントまでの自動化されたコンプライアンスワークフロー
- **Thematic Macro Advisor - 投資テーマ分析**: 新興機会の特定と統合投資戦略開発を伴うクロスポートフォリオテーマポジショニング

**主要機能**: 定量的ポートフォリオデータと定性的リサーチを組み合わせたマルチツールハイブリッド分析、リアルタイムイベントインパクト評価、マンデートコンプライアンス自動化、テーマ別投資戦略開発。

**追加シナリオ**: AIインフラストラクチャ戦略開発を含むThematic Macro Advisorの追加シナリオについては、[テーマアドバイザーシナリオ](demo_scenarios_thematic_advisor.md)も参照してください。

## このドキュメントの使用方法

### デモンストレーション用
1. コンテキスト設定のためにペルソナとビジネス課題をレビュー
2. 正確なユーザー入力を含むステップバイステップのデモフローに従う
3. 主要機能を強調するためにトーキングポイントを使用
4. ビジネスインパクトと技術的差別化要因についてシナリオラップアップを参照

### 開発用
- 各シナリオは必要なツールとデータソースを文書化
- 期待される応答はエージェント動作の検証基準を提供
- 技術的差別化要因はSnowflake固有の機能を強調
- 使用ツールセクションはエージェントツール設定にマッピング

### トレーニング用
- ビジネスコンテキスト設定はリアルなユーザーシナリオを提供
- マルチステップワークフローは完全なビジネスプロセスを実証
- シナリオラップアップはビジネス成果と競争優位性を強調
- 強調された主要機能セクションは実証可能な機能に焦点
