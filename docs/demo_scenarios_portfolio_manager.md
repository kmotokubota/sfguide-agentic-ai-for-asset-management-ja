# SAMデモ - ポートフォリオマネージャーシナリオ

ポートフォリオマネージャーロール向けの完全なデモシナリオ。ステップバイステップの会話、期待される応答、データフローを含みます。

---

## ポートフォリオマネージャー

### Portfolio Copilot - ポートフォリオインサイト＆ベンチマーキング

#### ビジネスコンテキスト設定

**ペルソナ**: Anna、Simulated Asset Managementのシニアポートフォリオマネージャー  
**ビジネス課題**: ポートフォリオマネージャーは、情報に基づいた投資決定を行うために、ポートフォリオ分析、保有情報、サポートリサーチへの即座のアクセスが必要です。従来のシステムでは複数のツール、手動のデータ収集、時間のかかる分析が必要で、重要な投資決定が遅れます。  
**価値提案**: 定量的な保有データと定性的なリサーチインサイトを数秒で組み合わせるAI駆動のポートフォリオ分析により、より迅速な意思決定とより良いリスク管理を実現。

**エージェント**: `portfolio_copilot`  
**利用可能データ**: 10ポートフォリオ、5,000証券、2,800リサーチドキュメント

#### デモフロー

**シーン設定**: Annaは週次ポートフォリオレビュー会議の準備をしており、Technology & Infrastructureポートフォリオのパフォーマンスを迅速に評価し、現在の保有を理解し、注意が必要な新たなリスクを特定する必要があります。

##### ステップ1: トップ保有銘柄の概要
**ユーザー入力**: 
```
SAM Technology & Infrastructureポートフォリオで市場価値による上位10保有銘柄は何ですか？
```

**使用ツール**:
- `quantitative_analyzer` (Cortex Analyst) - SAM_ANALYST_VIEWからポートフォリオ保有データをクエリ

**期待される応答**:
- テーブル表示: ティッカー、企業名、ウェイト%、市場価値USD
- 6.5%超のポジションにフラグ（集中警告）
- 上位10銘柄の合計エクスポージャー率

**トーキングポイント**:
- SQLや複雑なクエリなしで即座のポートフォリオ分析
- ビジネスルールに基づく自動的な集中リスクフラグ
- 遅延なしのデータウェアハウスからのリアルタイムデータ

##### ステップ2: トップ保有銘柄の最新リサーチ

**ユーザー入力**: 
```
先ほど表示されたトップ保有銘柄に基づいて、3つの最大ポジションについてブローカーリサーチは何と言っていますか？
```

**使用ツール**:
- `search_broker_research` (Cortex Search) - ステップ1で特定された上位3社のアナリストリサーチを検索

**期待される応答**:
- 箇条書き: 企業 → ステップ1からの上位3保有銘柄の最新レポートタイトルと日付
- 主要な投資テーマの簡潔な要約
- 格付け分布（買い/ホールド/売り）
- 大きなポジションサイズとリサーチセンチメントの整合性分析

##### ステップ3: セクターリスク評価

**ユーザー入力**: 
```
それらのトップ保有銘柄とそのリサーチを見て、このポートフォリオのセクター集中リスクは何ですか、特に最大ポジションを持つ企業について？
```

**使用ツール**:
- `quantitative_analyzer` (Cortex Analyst) - SAM_ANALYST_VIEWからセクター配分と集中リスクを分析

**期待される応答**:
- ステップ1からのトップ保有銘柄を強調したセクター配分内訳
- 大きなポジションを通じたセクターエクスポージャーを示す集中分析
- ベンチマークセクターウェイトとの比較
- ポジションサイズとセクター集中を組み合わせたリスク評価

##### ステップ4: 統合リスク＆アクションプラン

**ユーザー入力**: 
```
集中分析とリサーチ結果に基づいて、最大ポジションのどれが注意を必要とし、どのようなアクションを検討すべきですか？
```

**使用ツール**:
- `quantitative_analyzer` (Cortex Analyst) - 集中データを取得
- `search_policies` (Cortex Search) - 会社ポリシーから集中閾値を取得
- `search_broker_research` (Cortex Search) - ステップ2からのリサーチセンチメントを参照

**期待される応答**:
- ステップ1-3で特定されたトップ保有銘柄の統合リスク評価
- 集中リスク（ステップ3）とリサーチセンチメント（ステップ2）の組み合わせ
- 大きくかつセクター集中リスクを持つポジションの具体的な懸念
- 優先アクションプラン: 即座の注意が必要なポジション vs 監視

##### ステップ5: ポートフォリオ管理決定

**ユーザー入力**: 
```
完全な分析に基づいて、取るべきポートフォリオアクションの正確なポジションサイズ、タイムライン、ドル金額を含む具体的な実装計画を提供してください。
```

**使用ツール**:
- `implementation_analyzer` (Cortex Analyst) - 取引コスト、市場インパクト、実行タイムラインを計算
- `quantitative_analyzer` (Cortex Analyst) - 現在のポジションデータとポートフォリオメトリクスを取得
- `search_policies` (Cortex Search) - マンデート要件と承認プロセスを参照

**期待される応答**:
- **実装計画**:
  - 目標ウェイト付きの具体的なポジション削減推奨
  - 実行アプローチ（TWAP、タイムライン、市場インパクト考慮）
- **リスクバジェット分析**: トラッキングエラーとマンデート利用への影響
- **承認要件**: ポリシーに基づく委員会通知

#### シナリオラップアップ

**ビジネスインパクトサマリー**:
- **時間節約**: ポートフォリオ分析時間を数時間から数分に短縮
- **リスク管理**: 新興リスクと集中問題のプロアクティブな特定
- **決定品質**: 統合された定量的・定性的インサイトによる意思決定の強化
- **運用効率**: 複数のレガシーシステムと手動プロセスを単一インターフェースで置換

### Portfolio Copilot - リアルタイムイベントインパクト＆二次リスク検証

#### ビジネスコンテキスト設定

**ペルソナ**: Anna、Simulated Asset Managementのシニアポートフォリオマネージャー  
**ビジネス課題**: 主要なマクロ経済イベントや地政学的ショックが発生した場合、ポートフォリオマネージャーは直接エクスポージャーと間接エクスポージャー（サプライチェーン依存関係経由）の両方を即座に評価する必要があります。  
**価値提案**: 直接エクスポージャーとマルチホップサプライチェーン依存関係分析を組み合わせたイベント駆動型リスク評価。

**エージェント**: `portfolio_copilot`

#### デモフロー

**シーン設定**: Annaは台湾で半導体生産に影響を与える大地震のニュースを聞いたばかりです。

##### 9ツールオーケストレーションのイベント駆動リスク評価

**ユーザー入力**: 
```
台湾で半導体生産に影響を与える大地震が発生したと聞きました。以下をお願いします：
1. このイベントを確認し、影響を受けるセクターを特定
2. すべてのポートフォリオにおける台湾ベースのテクノロジー保有への直接エクスポージャーを表示
3. サプライチェーン依存関係を通じた間接エクスポージャーを計算（特にApple、NVIDIAなど）
4. 集中制限に違反するポジションがないか確認
5. SECデータを使用して最もエクスポージャーの高い企業の財務健全性を分析
6. サプライチェーンの回復力について最近の決算説明会で経営陣が何を言っているか確認
7. 半導体エクスポージャーについてアナリストが何を推奨しているか確認
8. 地理的集中とイベント対応に関する当社のポリシーを確認
9. このリスク評価を文書化した投資委員会メモを生成
```

**使用ツール**:
- `search_macro_events` (Cortex Search) - イベント確認
- `quantitative_analyzer` (Cortex Analyst) - 直接エクスポージャー分析
- `supply_chain_analyzer` (Cortex Analyst) - 間接エクスポージャー計算
- `compliance_checker` (Cortex Analyst) - 制限違反チェック
- `sec_financials_analyzer` (Cortex Analyst) - 財務健全性分析
- `search_company_events` (Cortex Search) - 経営陣コメンタリー
- `search_broker_research` (Cortex Search) - アナリスト推奨
- `search_policies` (Cortex Search) - ポリシー確認
- `memo_generator` - 投資委員会メモ生成

### Portfolio Copilot - AI支援マンデートコンプライアンス＆証券代替

#### ビジネスコンテキスト設定

**ペルソナ**: Anna、Simulated Asset Managementのシニアポートフォリオマネージャー  
**ビジネス課題**: ポートフォリオマネージャーは、集中違反を特定し、同様の特性を持つ代替証券を見つけ、最小限のトラッキングエラーで実行を計画する必要があります。  
**価値提案**: 違反検出から代替分析、委員会ドキュメントまでの自動化されたコンプライアンスワークフロー。

**エージェント**: `portfolio_copilot`

#### シナリオラップアップ

**ビジネスインパクトサマリー**:
- **コンプライアンス効率**: 自動違反検出とポリシー参照
- **代替品質**: ファクターマッチングと流動性分析
- **実行精度**: 市場インパクト最小化の取引計画
- **監査対応**: 完全なドキュメント生成

**技術的差別化要因**:
- **マルチツールオーケストレーション**: 単一クエリで4-9ツールを統合
- **リアルタイムコンプライアンス**: ポリシー閾値とポジションの即座比較
- **代替インテリジェンス**: ファクター類似性と流動性に基づく証券マッチング
